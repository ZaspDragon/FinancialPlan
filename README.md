# Financial Plan Advisor

A mobile-friendly GitHub Pages web app that acts like a simple personal financial advisor.

## What it does
- Tracks two incomes
- Splits shared bills 50/50 or proportionally
- Tracks personal bills for each person
- Shows true free cash after bills
- Recommends where monthly money should go
- Suggests allocations for emergency fund, investing, house fund, lifestyle, and debt payoff
- Saves data in the browser automatically with localStorage
- Lets you export and import your setup as JSON

## Files
- `index.html`
- `styles.css`
- `app.js`

## How to use on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and root folder.
6. Save.
7. Wait a minute and your site will be live.
